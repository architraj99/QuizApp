# QuizApp

QuizApp is a simple browser  quiz game builtby me
Players will have to answer multiple choice questions,
earn points for correct answers,
and receive a final score at the end of the quiz.

## Don"t worry ... there are alot easy questions 

## Features

- Clean page look aesthetic
- a lot of multiple choice questions are given
- Score tracking according to your perfomance
- Question progress indicator where you can see how many you attempted
- Countdown timer for each question
- Instant answer feedback
- Final result screen
- Restart quiz option

## How To Play

1. Click the Start Quiz button.
2. Read the question carefully... ( also you have limited time )
3. Select the answer you think is correct.
4. Move to the next question.
5. Continue until all questions are completed.
6. View your final score on the results screen.
7. Restart and play again anytime.
